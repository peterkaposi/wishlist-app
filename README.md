# Wishlist App
A modern wishlist application built with Next.js, TypeScript, Tailwind CSS, Lucide React and Zustand store.

## Live Demo
- GitHub Repository: https://github.com/peterkaposi/wishlist-app
- Vercel Deployment: https://wishlist-app-opal-pi.vercel.app

## Features
- Browse products from FakeStore API
- Add/remove products to/from wishlist
- View detailed product information
- Responsive design (mobile, tablet, desktop)
- LocalStorage persistence for wishlist
- Real-time wishlist count in navigation

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS
- Zustand
- FakeStore API
- Lucide React

## Installation
1. Clone the repository:
```bash
git clone https://github.com/peterkaposi/wishlist-app.git
cd wishlist-app
```
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production
```bash
npm run build
npm start
```

## Deploy to Vercel
```bash
npx vercel --prod
```
